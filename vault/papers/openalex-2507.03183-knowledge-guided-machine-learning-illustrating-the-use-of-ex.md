# Knowledge-Guided Machine Learning: Illustrating the use of Explainable Boosting Machines to Identify Overshooting Tops in Satellite Imagery

**Authors**: Nathan Mitchell, Lander Ver Hoef, Imme Ebert‐Uphoff, Kristina Moen, Kyle Hilburn, Yoonjin Lee, Emily J. King
**Date**: 2026-06-19
**Paper ID**: [openalex:2507.03183](https://arxiv.org/abs/2507.03183)

## Summary

This paper explores the application of Explainable Boosting Machines (EBMs) to enhance interpretability in severe weather forecasting. By extracting scalar features from satellite imagery using Knowledge-Guided Machine Learning techniques, the authors bridge the gap between complex meteorological data and interpretable models. The study showcases how EBMs enable human-machine collaboration, allowing domain experts to refine model strategies for identifying overshooting tops, thereby fostering more reliable and transparent decision-making in high-stakes meteorological tasks.

## Key Contributions

- Introduces Explainable Boosting Machines (EBMs) as an interpretable alternative to black-box models in meteorological applications.
- Demonstrates a Knowledge-Guided Machine Learning workflow by extracting scalar features (e.g., cloud texture via Gray-Level Co-occurrence Matrices) to enable EBM training on satellite imagery.
- Shows that EBMs allow for human-in-the-loop refinement, resulting in a model that aligns with meteorologist domain knowledge for overshooting top (OT) detection.
