# Kepler Image-Subtracted Light Curves and Variable Star Catalog of NGC 6819

**Authors**: Melinda Soares-Furtado, Rianna Kuenzi, Rachel Lee McClure, Nicholas T. Marston, Evan Linck, Robert D. Mathieu, Andrew Vanderburg, Joel D. Hartman, Kyle Cudworth, R. Gagliano, Thomas Jacobs, Martti H. Kristiansen, Mark Omohundro, Hans Martin Schwengeler
**Date**: 2026-06-23
**Paper ID**: [openalex:2606.24114](https://arxiv.org/abs/2606.24114)

## Summary

This study presents a high-precision catalog of variable stars in the NGC 6819 open cluster by processing Kepler superstamp time-series data using advanced image subtraction and reduction techniques. By leveraging Gaia DR3 astrometry, the authors disentangled cluster members from background stars and generated 81,498 high-fidelity light curves. This dataset enables detailed investigations into stellar evolution and angular momentum tracking, contributing 26 newly discovered periodic variable stars to the literature.

## Key Contributions

- Processed Kepler superstamp data for the NGC 6819 open cluster to generate 81,498 high-precision light curves across 11,055 sources.
- Achieved a best RMS precision of 33 ppm for stars in the 12-12.5 mag range, demonstrating the effectiveness of the applied image subtraction and reduction pipeline.
- Identified and classified 87 periodic variables in NGC 6819, 26 of which are new discoveries, and provided membership confirmation using Gaia DR3 astrometric data.
