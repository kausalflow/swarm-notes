# The Double-lined Eclipsing γ Doradus System AX Draconis in a 0.568-day Orbit

**Authors**: Jae Woo Lee, Min-Ji Jeong, Kyeongsoo Hong, Jang-Ho Park, Pakakaew Rittipruk, Hye-Young Kim
**Date**: 2026-06-04
**Paper ID**: [openalex:2606.05529](https://arxiv.org/abs/2606.05529)

## Summary

This paper presents a detailed analysis of the near-contact binary system AX Dra using high-resolution spectroscopy and TESS multi-cadence photometry. The researchers measure key stellar parameters and model the system as a semi-detached binary, demonstrating that the primary star is a γ Dor-type pulsator. The study concludes that AX Dra represents the shortest-period double-lined eclipsing binary known to harbor a γ Dor pulsator, with the primary likely evolving as an accretor due to mass transfer.

## Key Contributions

- Provides the first time-series spectroscopic analysis of the near-contact binary AX Dra using BOES echelle spectrograph data.
- Determines precise stellar parameters including masses, radii, and luminosities through joint modeling of radial velocities and TESS photometric data.
- Identifies AX Dra as the shortest-period double-lined eclipsing binary system containing a γ Dor-type pulsator.
