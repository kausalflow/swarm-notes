# Active learning-enabled multi-objective design of thermally conductive and mechanically compliant polymers

**Authors**: Yuhan Liu, Jiaxin Xu, Renzheng Zhang, Meng Jiang, Tengfei Luo
**Date**: 2026-03-24
**ArXiv**: [2603.23494](https://arxiv.org/abs/2603.23494)

## Summary

This work introduces an active learning (AL) framework built upon multi-objective Bayesian optimization (MOBO) to address the challenge of discovering polymers with high thermal conductivity and low mechanical modulus simultaneously. The methodology trains Deep Kernel Learning (DKL) surrogate models on initial molecular dynamics (MD) data to predict these two conflicting properties and their associated uncertainties. By employing the qNEHVI acquisition function, the system iteratively selects the most informative polymer candidates from a large database for MD evaluation, effectively navigating the design space. The approach successfully identified six materials on the Pareto front, demonstrating a significant reduction in development time and providing actionable guidance for designing multifunctional polymers.

## Key Contributions

- Development of an active learning (AL) framework powered by multi-objective Bayesian optimization (MOBO) to simultaneously optimize for high thermal conductivity (TC) and low bulk modulus in polymers.
- Integration of Deep Kernel Learning (DKL) surrogate models for TC and modulus prediction, trained on data generated by a high-throughput molecular dynamics (MD) pipeline.
- Utilization of the parallel noisy expected hypervolume improvement (qNEHVI) acquisition function for efficient, iterative screening of a large unlabeled polymer database.
- Identification of six polymer candidates on the Pareto front representing optimal trade-offs, validated through interpretability analysis and synthesizability assessment.

## Limitations

The framework's efficiency is still dependent on the quality of the initial MD-generated dataset and the computational cost of running the full MD pipeline for suggested candidates in each iteration. The interpretability analysis provides insights but does not fully substitute experimental validation for practical synthesis.

## Key Concepts

- [Active Learning for Multi-Objective Optimization](../concepts/active-learning-multi-objective-optimization.md): A machine learning framework that iteratively uses a surrogate model and an acquisition function to efficiently explore the design space for materials satisfying multiple conflicting objectives.
- [Deep Kernel Learning](../concepts/deep-kernel-learning.md): A surrogate modeling technique that uses deep neural networks to learn complex feature representations for use within a Gaussian Process framework, enabling better modeling of high-dimensional input data.
