# Foveated Diffusion: Efficient Spatially Adaptive Image and Video Generation

**Authors**: Brian Chao, Lior Yariv, Howard Xiao, Gordon Wetzstein
**Date**: 2026-03-24
**ArXiv**: [2603.23491](https://arxiv.org/abs/2603.23491)

## Summary

Foveated Diffusion addresses the computational bottleneck in high-resolution image and video generation by exploiting the non-uniform acuity of human vision, focusing detail only near the estimated gaze location. The method constructs a mixed-resolution token space guided by a foveation mask, allowing a pre-trained diffusion model to be efficiently adapted to this new representation. This non-uniform token allocation drastically reduces the computational load required for sampling, yielding perceptually equivalent results to full-resolution generation. The authors validate the practical effectiveness and scalability of this efficiency axis through rigorous testing and user studies.

## Key Contributions

- Introduction of Foveated Diffusion, a method to reduce generation complexity by allocating tokens non-uniformly based on a foveated resolution mask derived from estimated gaze location.
- Development of a mechanism to construct mixed-resolution tokens directly from high-resolution data, enabling post-training of existing diffusion models.
- Demonstration that the perceptually generated output is indistinguishable from full-resolution generation while achieving significant reductions in token count and generation time.
- Validation through extensive analysis and a user study confirming the practical scalability of foveation for efficient generative modeling.

## Limitations

The approach is contingent on the availability or reliable estimation of the user's gaze location (foveal region).
