# Predictive Analytics in E-Commerce for CustomerBehavior Forecasting using hybrid Ret-DNN withXGBoost Model

**Authors**: Degala Pushpa Sri, Mayank Atreya, Lakshmi. H, Navin Chhibber, Mukesh Soni
**Date**: 2026-06-16
**Paper ID**: [openalex:2606.17931](https://arxiv.org/abs/2606.17931)

## Summary

This paper proposes a hybrid architecture combining a Retail Deep Neural Network (Ret-DNN) and an Extreme Gradient Boosting (XGBoost) model to improve customer behavior forecasting. The Ret-DNN serves as a feature extractor to capture temporal and transactional contexts, while the XGBoost regressor predicts the final customer purchase probability. Evaluation on a UK-based retail dataset shows that the hybrid approach significantly outperforms a standalone Ret-DNN model.

## Key Contributions

- Introduces a hybrid Ret-DNN-XGBoost model architecture designed for customer behavior forecasting in retail environments.
- Utilizes Ret-DNN as a specialized temporal feature extractor for transactional data, followed by XGBoost for final purchase probability prediction.
- Demonstrates superior performance over standalone Ret-DNN models with a Mean Absolute Error (MAE) of 0.2193 on a large-scale UK e-commerce transaction dataset.
