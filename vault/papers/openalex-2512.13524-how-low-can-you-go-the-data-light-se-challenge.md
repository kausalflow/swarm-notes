# How Low Can You Go? The Data-Light SE Challenge

**Authors**: K. Ganguly, Tim Menzies
**Date**: 2026-06-30
**Paper ID**: [openalex:2512.13524](https://arxiv.org/abs/2512.13524)

## Summary

This paper challenges the prevailing reliance on massive datasets and compute-intensive optimizers in software engineering research by showing that simple, data-light methods frequently achieve near-optimal results. By evaluating over 100 tasks, the authors demonstrate that diversity sampling and minimal Bayesian learners often match the performance of complex state-of-the-art optimizers like SMAC or DEHB with significantly fewer samples. The study calls for a shift toward cost-efficient, rapid guidance models, introducing a formal framework to identify when minimal data requirements are sufficient for SE applications.

## Key Contributions

- Demonstrates that simple, data-light optimization methods achieve over 90% of SOTA performance across 100+ software engineering tasks using only a few dozen labels.
- Proposes the 'data-light challenge' for software engineering research, formalizing conditions under which minimal labeling and lightweight computation suffice for engineering needs.
- Provides a comprehensive repository containing lightweight baseline algorithms and a mathematical formalization of labeling efficiency for reproducibility and future benchmarking.
