# Towards exact temporal aggregation of time-coupled energy storage models via active constraint set identification and machine learning

**Authors**: Thomas Klatzer, David Cardona-Vasquez, Luca Santosuosso, Sonja Wogrin
**Date**: 2026-07-13
**Paper ID**: [openalex:2510.14451](https://arxiv.org/abs/2510.14451)

## Summary

This paper introduces a theoretical framework for exact temporal aggregation in energy storage models by utilizing active constraint set identification and dual variable analysis. To address practical implementation challenges, the authors incorporate machine learning classifiers and clustering to inform the selection of representative periods for renewable and storage co-scheduling. The proposed method enables significant computational efficiency in power systems optimization while maintaining high solution fidelity compared to full-scale reference models.

## Key Contributions

- Theoretical framework for exact temporal aggregation of energy storage systems using active constraint set identification and dual information.
- ML-based classification and clustering pipeline to approximate optimal temporal aggregation for co-scheduling renewables and storage.
- Demonstrates significant computational speed-ups in power system optimization compared to full-scale models without sacrificing solution accuracy.
