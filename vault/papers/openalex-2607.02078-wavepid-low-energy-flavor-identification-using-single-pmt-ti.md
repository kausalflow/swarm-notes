## Summary

The paper introduces WavePID, a new classifier for neutrino flavor identification in the IceCube Neutrino Observatory, specifically designed to address challenges in the 5-100 GeV regime. By leveraging nanosecond-scale pulse timing on individual detector modules, WavePID extracts complementary features based on Cherenkov emission geometry that morphology-based methods miss. When integrated with existing graph neural network classifiers, it provides measurable improvements in cascade purity and classification performance. These results suggest that per-module timing is a critical, underutilized observable for future low-energy physics reconstruction.

## Key Contributions

- Introduces WavePID, a template-based log-likelihood-ratio classifier that utilizes nanosecond-scale pulse timing.
- Demonstrates that per-module timing information provides complementary features for neutrino flavor identification alongside morphology-based graph neural networks.
- Improves cascade purity and overall classification performance in the 5-100 GeV energy regime compared to state-of-the-art neural network baselines.
