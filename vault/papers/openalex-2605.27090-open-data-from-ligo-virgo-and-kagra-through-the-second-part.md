# Open Data from LIGO, Virgo, and KAGRA through the Second Part of the Fourth Observing Run

**Authors**: The LIGO Scientific Collaboration, The LIGO Scientific Collaboration, The LIGO Scientific Collaboration
**Date**: 2026-05-26
**Paper ID**: [openalex:2605.27090](https://arxiv.org/abs/2605.27090)

## Summary

This paper details the public release of gravitational-wave observatory data from the second half of the fourth observing run (O4b) and the ER16 engineering run via the Gravitational Wave Open Science Center. The release provides high-fidelity calibrated strain time series alongside auxiliary noise-subtraction channels for the international LIGO-Virgo-KAGRA network. These data products are essential for ongoing research in gravitational-wave astronomy and serve as the foundation for the upcoming version 5.0 of the Gravitational-Wave Transient Catalog.

## Key Contributions

- Provides public access to calibrated strain time series from the O4b observing run (April 2024 to January 2025).
- Integrates data from the ER16 engineering run to support detector characterization and noise subtraction research.
- Updates the GWOSC portal with new analysis products corresponding to version 5.0 of the Gravitational-Wave Transient Catalog.
