# Guardrail Selection in Line Charts to Contextualize Persuasive Visualizations

**Authors**: Khandaker Abrar Nadib, Marina Kogan, Alexander Lex, Maxim Lisnic
**Date**: 2026-06-17
**Paper ID**: [openalex:2605.19017](https://arxiv.org/abs/2605.19017)

## Summary

This paper addresses the problem of misleading, cherry-picked persuasive visualizations on social media by proposing guardrail sampling strategies that integrate contextual comparison lines directly into line charts. Through a mixed-design study using COVID-19 and stock market data, the authors evaluate how these guardrails influence user perception, trust, and accuracy. Results indicate that embedding contextual information improves users' performance judgments and overall assessment of data completeness compared to unguided visualizations.

## Key Contributions

- Introduces a set of guardrail sampling strategies for embedding contextual comparison lines into line charts to mitigate misleading persuasive visualizations.
- Demonstrates through a preregistered mixed-design study that these guardrails significantly improve user trust, performance judgment accuracy, and perceived context completeness.
- Provides empirical evidence and design insights into user preferences for contextual aids in time-series data exploration.
