# Warp Quantification Analysis: A Framework for Path-Based Signal Alignment Metrics

**Authors**: Sir-Lord Wiafe, Vince D. Calhoun
**Date**: 2026-04-21
**Paper ID**: [openalex:2509.14994](https://arxiv.org/abs/2509.14994)

## Summary

This paper introduces Warp Quantification Analysis (WQA), a framework designed to extract rich geometric and structural descriptors from Dynamic Time Warping (DTW) paths. Unlike traditional approaches that collapse alignment information into a single scalar distance, WQA provides a multifaceted view of temporal coupling. The authors demonstrate that these metrics effectively isolate different drivers of temporal variation in simulations and reveal clinically relevant network signatures in fMRI data. This method offers a generalizable approach for characterizing nonlinear alignment in various scientific domains.

## Key Contributions

- Introduced Warp Quantification Analysis (WQA), a framework for extracting geometric descriptors from DTW paths.
- Demonstrated via simulations that WQA metrics exhibit minimal crosstalk in tracking distinct temporal drivers.
- Validated the utility of WQA in fMRI analysis by identifying network signatures correlated with schizophrenia negative symptom severity.
