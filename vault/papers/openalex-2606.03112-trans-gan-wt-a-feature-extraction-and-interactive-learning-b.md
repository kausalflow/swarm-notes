# Trans GAN-WT: A Feature Extraction and Interactive Learning-Based Anomaly Detection Model for Wind Turbine Time Series Data

**Authors**: Jingzhe Kang
**Date**: 2026-06-02
**Paper ID**: [openalex:2606.03112](https://arxiv.org/abs/2606.03112)

## Summary

This paper introduces TransGAN-WT, an anomaly detection model for wind turbine data that combines Transformer-based feature extraction with generative adversarial networks. The framework employs an autoregressive inference mechanism for multimodal feature extraction and a dedicated module for interactive, multi-scale temporal learning to handle data redundancy. Experimental results on real wind turbine datasets demonstrate significant improvements in detection accuracy and reliability compared to existing state-of-the-art methods.

## Key Contributions

- Introduced TransGAN-WT, a fusion model of Transformer and GAN architectures, for anomaly detection in wind turbine operational data.
- Developed a temporal feature extraction module that enables interactive learning between multi-scale features to mitigate time redundancy.
- Achieved a 96.10% average F1 score on wind turbine datasets, outperforming state-of-the-art baselines with a 0.06% false positive rate.
