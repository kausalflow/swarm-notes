# An astrometric search for planets in debris disk systems

**Authors**: Elisabeth M. Penderghast, Benjamin C. Bromley, Scott J. Kenyon, Joan Najita
**Date**: 2026-04-08
**Paper ID**: [openalex:2604.07068](https://arxiv.org/abs/2604.07068)

## Summary

This paper investigates the potential for detecting planetary companions in debris disk systems using astrometric data from the Gaia Data Release 3 (DR3) archive. By training a machine learning model on Gaia parameters from known exoplanetary host stars, the authors evaluate 176 debris disk-hosting stars to identify those likely to harbor undiscovered planets. The study confirms that metrics like the Gaia RUWE parameter are effective at detecting massive companions, setting the stage for future time-series analyses with the upcoming Gaia DR4 dataset.

## Key Contributions

- Identified promising debris disk systems for planet detection by leveraging Gaia DR3 astrometric parameters.
- Validated the sensitivity of the Gaia RUWE (Renormalized Unit Weight Error) parameter and a machine-learning-derived metric to the presence of planetary-mass companions.
- Established a comparative framework using known exoplanetary host stars to filter and rank candidate stars with debris disks for future Gaia DR4 time-series follow-up.
