# System Level Analysis and Management of Orbital Debris Using Empirical Dynamic Modeling

**Authors**: Asaad S. Abdul-Hamid, Hao Chen
**Date**: 2026-05-21
**Paper ID**: [openalex:2605.21892](https://arxiv.org/abs/2605.21892)

## Summary

This paper addresses the orbital debris challenge by framing it as a complex dynamical system rather than a collection of independent variables. The authors employ Empirical Dynamic Modeling to reconstruct a shadow attractor of the system, enabling the analysis of system-level causality from limited observable data. This approach provides a robust mechanism to simulate the potential future consequences of various space policies on debris mitigation.

## Key Contributions

- Reconstructed a shadow attractor of the orbital debris system using limited observable variables (debris counts, object counts, launch rates).
- Established a causal framework for modeling the orbital debris system as a complex dynamic process rather than independent variables.
- Enabled scenario-based simulation of policy impacts on long-term orbital debris accumulation using the reconstructed manifold.

## Key Concepts

- [[empirical-dynamic-modeling]]: A non-parametric approach for identifying the underlying manifold structure of a dynamical system from short, noisy time-series data.
