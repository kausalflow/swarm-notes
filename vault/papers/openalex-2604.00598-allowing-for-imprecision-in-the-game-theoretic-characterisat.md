# Allowing for imprecision in the game-theoretic characterisation of the Poisson process

**Authors**: Alexander Erreygers
**Date**: 2026-04-01
**Paper ID**: [openalex:2604.00598](https://arxiv.org/abs/2604.00598)

## Summary

This paper extends the classical game-theoretic definition of the Poisson process to account for rate imprecision. Instead of assuming an exact, known rate, the author defines a process based on bounds, leading to an 'imprecise Poisson process'. This framework allows for more realistic modeling of point processes when information about the intensity is limited or uncertain. The resulting model maintains essential theoretical properties similar to the precise Poisson process while offering greater flexibility for practical applications.

## Key Contributions

- Formalizes an 'imprecise Poisson process' within a game-theoretic framework by replacing exact rate assumptions with rate bounds.
- Demonstrates that the imprecise Poisson process retains key theoretical properties of the classical precise Poisson process.
- Extends Vovk's (1993) game-theoretic analysis of point processes to handle more realistic, imprecise information settings.
