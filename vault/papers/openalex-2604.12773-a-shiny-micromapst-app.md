# A Shiny micromapST App

**Authors**: Randall Powers, Wendy Martinez
**Date**: 2026-04-14
**Paper ID**: [openalex:2604.12773](https://arxiv.org/abs/2604.12773)

## Summary

This paper introduces a graphical user interface (GUI) developed in R Shiny to simplify the generation of linked micromaps, which are effective alternatives to traditional choropleth maps for displaying spatial statistical summaries. By abstracting the complex data frame requirements of the original micromapST R package, the application enables more intuitive and efficient exploration of geographically indexed data. The paper functions as a tutorial, demonstrating the tool's utility through real-world applications involving various statistical visualizations, including time series and scatterplots.

## Key Contributions

- Introduces a R Shiny graphical user interface to streamline the creation of linked micromaps using the micromapST package.
- Demonstrates that interactive visualization significantly lowers the barrier to entry for generating complex linked micromaps compared to manual data frame configuration.
- Provides a tutorial on visualizing geographically indexed statistical data, addressing challenges in formatting and layout design for spatial summaries.
