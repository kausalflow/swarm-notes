# Time Series Analysis in Machine Learning

**Authors**: A. Pagliaro, A. Anzalone
**Date**: 2026-06-10
**Paper ID**: [openalex:2606.11746](https://arxiv.org/abs/2606.11746)

## Summary

This paper provides a comprehensive pedagogical review of time series analysis, bridging the gap between classical statistical models and modern machine learning techniques. It systematically covers foundational concepts like stationarity and seasonality alongside established methods such as ARIMA and state-space models. Furthermore, it details the adoption of deep learning architectures, including recurrent, convolutional, and transformer-based models, providing a unified perspective for researchers applying these methods to diverse domains.

## Key Contributions

- Provides a pedagogical synthesis of foundational time series concepts and classical statistical models, establishing a baseline for machine learning approaches.
- Reviews the transition from traditional autoregressive and state-space methods to modern deep learning architectures, including RNNs, CNNs, and transformers.
- Integrates theoretical frameworks with cross-disciplinary practical applications, highlighting shared principles across astronomy, meteorology, and finance.
