# Differential Privacy of Network Parameters From a System Identification Perspective

**Authors**: Andrew R. Campbell, Anna Scaglione, Hang Liu, Victor Daniel Elvira, Sean Peisert, Daniel Arnold
**Date**: 2026-04-21
**Paper ID**: [openalex:2509.20460](https://arxiv.org/abs/2509.20460)

## Summary

This paper investigates the vulnerability of network graph shift operators (GSO) to system identification (SI) attacks when sharing cyber-physical simulation outputs. The authors model these outputs as time-series generated by graph filters driven by differentially private inputs, identifying the theoretical conditions that prevent an analyst from recovering the underlying network structure. By linking (ϵ, δ)-differential privacy bounds to the spectral properties of the filter and the noise covariance, they demonstrate how to balance data privacy with analytical utility.

## Key Contributions

- Establishes a formal link between (ϵ, δ)-differential privacy bounds and the spectral characteristics of graph filters used in cyber-physical systems.
- Proves that smooth graph filters and low-condition-number input noise covariance enhance privacy against system identification attacks on graph shift operators.
- Frames the problem of protecting network topology during simulation sharing as an inverse system identification problem.
