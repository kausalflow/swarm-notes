# OccAny: Generalized Unconstrained Urban 3D Occupancy

## Summary

OccAny addresses the scalability and generalization limitations of existing 3D occupancy prediction methods, which rely heavily on in-domain annotations and precise sensor priors. The model is presented as the first generalized framework capable of predicting metric occupancy coupled with segmentation features from uncalibrated, out-of-domain inputs, including sequential, monocular, or surround-view images. Key technical innovations include Segmentation Forcing, which enhances occupancy quality via mask-level supervision, and a Novel View Rendering pipeline for test-time view augmentation to improve geometry completion. Experiments show OccAny surpasses visual geometry baselines and remains competitive with in-domain self-supervised methods on urban occupancy prediction tasks.

## Key Contributions

- Introduction of OccAny, the first generalized 3D occupancy framework capable of predicting metric occupancy and segmentation features from uncalibrated, out-of-domain, and sequential/monocular/surround-view inputs.
- Proposal of Segmentation Forcing to improve the quality of predicted 3D occupancy while simultaneously enabling fine-grained mask-level prediction.
- Development of a Novel View Rendering pipeline that infers novel-view geometry to serve as a test-time view augmentation mechanism for enhanced geometry completion.
- Demonstration that OccAny outperforms existing visual geometry baselines across three input settings on established urban occupancy prediction benchmarks.

## Key Concepts

- [Segmentation Forcing](../concepts/segmentation-forcing.md): A technique proposed in OccAny that enhances occupancy prediction quality by enabling mask-level prediction through the integration of segmentation information.
- [Novel View Rendering Pipeline](../concepts/novel-view-rendering-pipeline.md): A novel pipeline within OccAny that infers novel-view geometry to facilitate test-time augmentation for improving the completion of 3D geometry.
