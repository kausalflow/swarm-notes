# Towards Reliable Zero-Shot Crowd Forecasting: Evaluating Time Series Foundation Models for Special Event Pedestrian Forecasting

**Authors**: Ziteng Li, Yanan Xin, Tina Comes, Serge Hoogendoorn
**Date**: 2026-07-20
**Paper ID**: [openalex:2607.17758](https://arxiv.org/abs/2607.17758)

## Summary

This paper investigates the reliability of pretrained time series foundation models for zero-shot probabilistic crowd forecasting during infrequent special events, addressing data scarcity and short observation windows. Using decision-oriented metrics tailored to short-duration events, the study evaluates foundation models on pedestrian-flow forecasting using the SAIL2025 event case. The findings distill practical insights regarding when zero-shot probabilistic forecasts and uncertainty quantification remain operationally reliable for crowd management.

## Key Contributions

- Investigates pretrained time series foundation models for zero-shot probabilistic crowd forecasting during infrequent special events without local retraining.
- Assesses models using decision-oriented metrics tailored to short-duration events, capturing sudden volatility and tail risks.
- Provides practical operational insights and reliability boundaries for crowd managers utilizing zero-shot pedestrian-flow forecasts.
