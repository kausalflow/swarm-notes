# Design principles for optoelectronic light-scattering reservoir computing at the edge of chaos

**Authors**: G KIM, YongKeun Park
**Date**: 2026-05-26
**Paper ID**: [openalex:2605.26848](https://arxiv.org/abs/2605.26848)

## Summary

This paper develops a systematic design framework for optoelectronic light-scattering reservoirs by mapping performance across three physical control axes: dynamics, input coupling, and interconnectivity. The authors identify a quantitative optimum along each axis, confirming that maximum memory capacity is achieved at the edge-of-chaos, marked by a near-zero maximal Lyapunov exponent. These findings extend critical phenomena observed in electronic reservoirs to photonic substrates and provide a transferable methodology for tuning reconfigurable physical hardware. The effectiveness of this framework is demonstrated through competitive performance on spoken-digit classification and chaotic time-series prediction.

## Key Contributions

- Established a three-axis design framework (reservoir dynamics, input coupling, interconnectivity) for reconfigurable optoelectronic light-scattering reservoirs.
- Demonstrated that memory capacity peaks at the edge-of-chaos, confirmed by near-zero maximal Lyapunov exponents in photonic hardware.
- Achieved 84.5% accuracy on the Speech Commands dataset and successful free-running Mackey-Glass time-series prediction using the identified optimal parameters.

## Key Concepts

- [[edge-of-chaos-reservoir-optimization]]: A framework for optimizing physical reservoirs by aligning control parameters to the edge-of-chaos regime characterized by near-zero maximal Lyapunov exponents.
