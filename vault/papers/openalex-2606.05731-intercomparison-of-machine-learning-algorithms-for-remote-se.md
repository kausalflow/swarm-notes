# Intercomparison of Machine Learning Algorithms for Remote Sensing-based In-season Crop Mapping

**Authors**: August Posch, Jitendra Kumar, Forrest M. Hoffman, Auroop R. Ganguly
**Date**: 2026-06-04
**Paper ID**: [openalex:2606.05731](https://arxiv.org/abs/2606.05731)

## Summary

This study addresses the lack of real-time in-season crop mapping by evaluating ten machine learning algorithms on their ability to classify crop types from satellite imagery before harvest. Using Harmonized Landsat-Sentinel surface reflectance time series combined with historical crop rotation data, the research achieves reliable mapping for corn and almonds by early June. The results highlight Support Vector Machines as the top-performing model and emphasize the critical role of accounting for interannual variability in agricultural remote sensing.

## Key Contributions

- Provides a rigorous intercomparison of ten machine learning algorithms for in-season crop mapping, identifying Support Vector Machines as the most effective baseline.
- Demonstrates accurate mapping of corn in Iowa and almonds in California by early June using Harmonized Landsat-Sentinel (HLS) imagery and historical crop rotation data.
- Quantifies the significant impact of interannual variability on model performance using year-wise cross-validation, establishing a robust framework for benchmarking in-season agricultural monitoring.
