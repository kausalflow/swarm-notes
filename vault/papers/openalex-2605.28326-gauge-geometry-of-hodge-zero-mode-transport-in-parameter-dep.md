# Gauge Geometry of Hodge Zero-Mode Transport in Parameter-Dependent Topological Data Analysis

**Authors**: Satoshi Kanno, Rei Nishimura, Hiroshi Yamauchi, Yoshi-aki Shimada
**Date**: 2026-05-27
**Paper ID**: [openalex:2605.28326](https://arxiv.org/abs/2605.28326)

## Summary

This paper introduces a framework to analyze parameter-dependent topological data by representing homological features as zero modes of the combinatorial Hodge Laplacian. By tracking these features in a common ambient chain space, the authors derive curvature and holonomy descriptors that effectively capture feature reorganization and accumulated memory over time. Numerical experiments demonstrate that this geometric approach detects tracking instabilities and differentiates systems with similar persistence diagrams that remain indistinguishable by static methods.

## Key Contributions

- Introduces a framework representing homological features as zero modes of the combinatorial Hodge Laplacian to track feature reorganization over changing parameters.
- Proposes curvature and holonomy as novel descriptors for local topological reorganization and cycle-level memory in dynamic data.
- Demonstrates that zero-mode transport captures tracking instabilities and cycle-level dependencies that are inaccessible to standard pointwise persistence diagram matching.

The identified open question addresses a fundamental limitation in spectral-based geometric frameworks—the breakdown of stability near singular points—which is a critical barrier for broader application in dynamic systems.
