# Bridging Deep Learning and Integer Linear Programming: A Predictive-to-Prescriptive Framework for Supply Chain Analytics

**Authors**: Khai Banh Nghiep, Duc Nguyen Minh, Lan Hoang Thi
**Date**: 2026-04-02
**Paper ID**: [openalex:2604.01775](https://arxiv.org/abs/2604.01775)

## Summary

This paper introduces a three-stage analytical framework that bridges demand forecasting with operational prescriptive analytics in supply chain management. The framework evaluates deep learning models, specifically N-BEATS and N-HiTS, against statistical benchmarks using a large-scale retail transaction dataset. N-BEATS is identified as the optimal forecasting model and integrated into an integer linear programming (ILP) pipeline to minimize delivery times under budget and capacity constraints. This approach demonstrates a practical workflow for translating accurate time-series projections into actionable, cost-effective logistical decisions.

## Key Contributions

- Proposed a three-stage predictive-to-prescriptive framework for supply chain logistics, integrating deep learning forecasting with integer linear programming.
- Demonstrated the superior forecasting performance of N-BEATS and N-HiTS over statistical baselines on a proprietary retail transaction dataset.
- Achieved cost-optimal and feasible shipping plans by using N-BEATS output for next-4-week demand to constrain an ILP optimization model.
