title: Translate static to deployed performance

**Background:** Impermanent provides a novel evaluation protocol for time series foundation models that contrasts with traditional static benchmarks by focusing on temporal generalization in dynamic environments.

**Question / Future Work:** A broader goal for the live benchmark framework is to serve as a shared resource for determining whether the good performance achieved by forecasting models in static, controlled settings successfully translates into reliable performance when deployed in dynamic, real-world conditions.
