title: Non-normal inference uncertainty quantification
---

**Background:** The inference of non-normal response geometry from multivariate time series relies on estimating local linear operators from finite windows of data, which inherently involves bias-variance trade-offs.

**Question / Future Work:** Establishing a systematic framework for quantifying the uncertainty associated with inferred response diagnostics, such as the non-normality ratio R, is necessary for distinguishing physically significant dynamical changes from estimation artifacts caused by finite-sample noise.

**Why It Matters:** Rigorous uncertainty quantification is essential for transitioning this diagnostic tool from an illustrative mechanism for exploratory analysis to a robust inferential framework capable of supporting scientific decision-making.

**Evidence:** Second, the framework should be complemented by uncertainty estimates for the inferred response plane Q, the reduced ratio R, the reaction support S(r^), and the eigenvalue splitting Δ.
