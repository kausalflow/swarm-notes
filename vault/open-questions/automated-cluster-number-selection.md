title: Automated Cluster Count Selection
---

**Background:** The selection of the optimal number of clusters in a mixture model remains a fundamental model selection challenge in unsupervised and semi-supervised learning.

**Question / Future Work:** Establishing principled, automated approaches for determining the optimal cluster count is essential, particularly by exploring prior distributions on the number of clusters to allow simultaneous estimation rather than relying on iterative fitting and comparison.

**Why It Matters:** Automating cluster count selection is critical for reducing researcher bias and improving the robustness of model selection in complex environmental epidemiology.

**Evidence:** A more sophisticated option would be to assign a prior distribution to C and estimate the optimal number of clusters simultaneously.
