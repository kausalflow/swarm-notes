title: Heterogeneous Graph Learning for 6G
---

**Background:** Non-terrestrial networks (NTN) integrate LEO satellites with diverse infrastructure, such as terrestrial base stations and high-altitude platforms, requiring cohesive management of a multi-technology graph.

**Question / Future Work:** There is a need to evolve current GNN-based satellite network frameworks into heterogeneous graph learning systems (HeteroGNNs) that can simultaneously represent and optimize diverse network nodes and link types within a cohesive 6G NTN framework.

**Why It Matters:** As satellite networks become core components of 6G, the inability to model heterogeneous network components as a unified graph will hinder seamless cross-tier resource management and end-to-end performance optimization.
