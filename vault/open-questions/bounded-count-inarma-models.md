title: Bounded count INARMA models
---

**Background:** Integer-valued autoregressive moving-average (INARMA) models for random fields primarily address unbounded count data in the non-negative integers.

**Question / Future Work:** There is a need to develop INARMA-like models specifically for bounded count data, where the state space is constrained to a finite set {0, ..., N}, as current models are not suitable for this domain.

**Why It Matters:** Many practical applications involve bounded counts that cannot be accurately represented by unbounded INARMA models.
