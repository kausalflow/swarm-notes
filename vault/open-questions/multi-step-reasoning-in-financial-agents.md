title: Multi-step financial agent reasoning
---

**Background:** Agentic systems for financial time-series analysis often rely on single-step query-to-function mappings, failing to support complex multi-step reasoning chains.

**Question / Future Work:** There is a need to develop multi-step agent architectures that can orchestrate sequences of external tool calls to resolve complex, compound financial analytical queries.

**Why It Matters:** Multi-step composition is critical for moving from simple function-calling assistants to autonomous financial research agents.
