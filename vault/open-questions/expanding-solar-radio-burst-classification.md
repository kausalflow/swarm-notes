title: Expand Solar Radio Burst Classification
---

**Background:** The development of systems for real-time analysis of solar radio data aims to improve space weather monitoring and accelerate scientific discovery.

**Question / Future Work:** Future work should focus on expanding the burst classification capabilities of the detection module beyond Type III and Type IIIb bursts to include other significant solar radio phenomena like Type II bursts (key signatures of CME-driven shocks) and Type V bursts, as well as noise-storm/background scenarios. This expansion requires the creation and use of a larger, multi-type labeled dataset for retraining or fine-tuning the existing model.
