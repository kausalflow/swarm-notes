title: Signed integer random fields
---

**Background:** Existing INARMA random field models are defined for non-negative integers, limiting their application to counting processes that cannot take negative values.

**Question / Future Work:** Developing INARMA models for random fields with support on the set of all integers (Z) remains an unresolved methodological challenge.

**Why It Matters:** Extension to signed integers is necessary for spatial data representing net differences or changes that allow negative values.
