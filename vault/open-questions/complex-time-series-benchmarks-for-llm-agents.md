title: Complex time series benchmarks for LLM agents
---

**Background:** Current evaluation methods for LLM-based time series analysis rely on constrained multiple-choice question answering, which limits the assessment of complex analytical capabilities. Tasks such as building analysis pipelines from scratch, handling data quality issues, and generating free-form interpretations remain under-explored.

**Question / Future Work:** There is a need to develop more comprehensive, open-ended benchmarks for time series analysis that go beyond multiple-choice formats to better evaluate an agent's ability to act as a complete data analyst.

**Why It Matters:** Developing more complex, realistic benchmarks is essential for determining if LLM agents can genuinely perform autonomous time series analysis in professional, open-world settings rather than just answering constrained exam questions.

**Evidence:** Benchmarks that require agents to write analysis pipelines from scratch, produce free-form interpretations, or deal with noisy and incomplete data would more accurately test whether agents can replace a human analyst.
