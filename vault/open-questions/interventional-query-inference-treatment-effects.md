title: Interventional Query Inference

**Background:** The presented model focuses on predicting the next clinical visit's events, which is a powerful, single-step predictive framework.

**Question / Future Work:** A key area for future work is exploring how to utilize the model's generative capabilities for more complex, multi-step causal inference, such as estimating the effects of potential new treatment policies by imitating interventional queries. This would require developing methods to query the model about counterfactual scenarios beyond simple observational prediction.
