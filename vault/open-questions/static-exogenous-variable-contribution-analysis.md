title: Static Exogenous Variable Contributions
---

**Background:** A key component of the PatchDecomp model is the set of input patches derived from time series data, which are utilized to generate predictions through an encoder-decoder structure.

**Question / Future Work:** The current model is unable to present the contributions of static exogenous variables to the predictions, which is planned as a subject for future investigation.
