title: Synchronous System Inference Limits
---

**Background:** Phase-locked systems present distinct analytical challenges for coupling inference because their dynamics contain limited information about the underlying interaction compared to asynchronous regimes. Although circle-map methods have shown promise for noisy synchronous oscillators, the theoretical justification for using reconstructed phases in this context remains an open question.

**Question / Future Work:** While this framework provides a basis for asynchronous systems, establishing the formal efficacy of circle-map-based inference methods for phases reconstructed from synchronous regimes is an open problem.

**Why It Matters:** Robust inference for synchronous systems is crucial for analyzing real-world neural and biological networks.

**Evidence:** Such inference is known to be challenging because nearly phase-locked oscillators provide only limited information about the underlying coupling... leaving it unclear whether phases reconstructed from data obey the same description.
