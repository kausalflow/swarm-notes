title: Emulator Data Assimilation Integration
---

**Background:** Deep learning emulators are increasingly utilized to replace costly numerical simulations, but a major obstacle is the seamless integration of these emulators with data assimilation systems to allow for state correction during operational forecasting.

**Question / Future Work:** Developing methodologies to integrate deep learning emulators within data assimilation frameworks is a key unresolved challenge, necessary for the systematic correction of the ocean state in operational forecasting systems.

**Why It Matters:** Without integration with data assimilation, emulators cannot effectively correct for initial state biases or incorporate real-time observations, which are essential for operational forecasting reliability.

**Evidence:** A key remaining research challenge is the integration of these emulators with data assimilation techniques, enabling systematic correction of the ocean state within operational forecasting systems.
