title: Combining Coding and Research Agents
---

**Background:** Automated machine learning (AutoML) systems that generate code rely on prompts to guide the LLM for task execution.

**Question / Future Work:** Investigating how to effectively combine coding agents (which generate code) and deep research agents (which provide domain knowledge) into a unified framework to generate expert-level algorithms warrants further study, as current approaches rely on injecting knowledge primarily through static base prompts.
