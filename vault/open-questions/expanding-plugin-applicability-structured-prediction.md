**Background:** A model-agnostic plugin for time series forecasting enhances base models by performing spectral decomposition on channel-patch dependencies via an adaptive routing mechanism.

**Question / Future Work:** The applicability of this method should be explored for a wider range of structured prediction tasks beyond time series forecasting, including spatiotemporal event forecasting and multivariate anomaly detection.
