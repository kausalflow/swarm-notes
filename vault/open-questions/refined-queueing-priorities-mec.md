title: Refining Request Scheduling Policies
---

**Background:** When managing queues for different request types at an edge server (EC), a preemptive priority discipline is often used to schedule service across queues.

**Question / Future Work:** Investigating more refined scheduling policies than the current preemptive priority discipline used between FIFO queues for service lifecycle requests and user requests, which could potentially impact service times and overall performance.
