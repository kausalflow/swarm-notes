title: Sharper Theory for Signature Dictionaries
---

**Background:** The number of path signature coordinates grows exponentially with the path dimension and truncation level, leading to high-dimensional dictionaries that require robust regularization.

**Question / Future Work:** Develop sharper statistical theory and adaptive feature selection methods for structured, sparsified, and high-dimensional signature dictionaries where the feature dimension drastically exceeds the sample size.

**Why It Matters:** Sharper high-dimensional theory for structured signature dictionaries is essential for controlling estimation error and mitigating overfitting in complex real-world path regression tasks.

**Evidence:** Second, because the number of signature coordinates grows quickly with the path dimension and truncation level, sharper theory is needed for structured, selected, and sparsified signature dictionaries.
