title: Ultra-long Sequence Memory Compression
---

**Background:** Time series foundation models frequently rely on attention mechanisms or memory structures that can scale poorly when processing extremely long sequences.

**Question / Future Work:** Future research must focus on investigating more aggressive memory compression techniques for dual-memory or state-space architectures to enable the efficient processing of ultra-long time series sequences containing millions of time steps.

**Why It Matters:** The ability to handle very long sequences is a major barrier for foundation models in many industrial and scientific applications.
