title: Holistic Interpretable Temporal Extraction
---

**Background:** Deep learning frameworks for time series classification frequently separate feature extraction from decision-making, which can limit the holistic interpretability of the model.

**Question / Future Work:** While the proposed framework improves interpretability at the feature-to-decision stage through prototype similarity, the underlying temporal feature extraction process remains relatively opaque. Future work could integrate the prototype-guided framework with existing temporal explanation techniques to create a more comprehensive and end-to-end interpretable model.

**Why It Matters:** Establishing a transparent link between raw temporal features and decision outcomes is critical for high-stakes domains such as healthcare and industrial monitoring, where simply explaining the final decision is insufficient.

**Evidence:** Second, while PDFTime enhances interpretability at the feature-to-decision stage through prototype similarity, the temporal feature extraction process itself remains implicit. Combining the proposed framework with complementary temporal explanation techniques is a potential extension.
