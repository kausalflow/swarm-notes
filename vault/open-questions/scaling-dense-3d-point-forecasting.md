title: Scaling Dense 3D Point Forecasting
---

**Background:** Vision-language models currently face memory and token-length limitations when representing fine-grained object geometry via 3D point trajectories.

**Question / Future Work:** Investigating architectures that can handle high-density point trajectories for complex or deformable objects without the memory and token-length constraints inherent in existing vision-language model backbones.

**Why It Matters:** Addresses a fundamental bottleneck in the fidelity of 3D motion forecasting, which currently limits the ability to model fine-grained and deformable object interactions.

**Evidence:** This sparse point setting is not sufficient to densely represent object geometry, limiting the model’s understanding of fine-grained structure and complex deformable motion.
