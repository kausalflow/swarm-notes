title: LLM Sentiment Predictive Validity
---

**Background:** In financial forecasting models, large language model (LLM)-generated sentiment scores are frequently integrated with numerical market data to capture potential predictive signals.

**Question / Future Work:** It remains unresolved whether the inclusion of LLM-generated sentiment features genuinely improves the predictive accuracy of financial models or if these features introduce noise that can degrade performance, warranting further validation against human-annotated ground truth.

**Why It Matters:** This is technically critical because many modern financial forecasting pipelines rely on LLM-driven sentiment extraction; establishing whether this actually adds predictive value or simply noise is essential for model reliability.

**Evidence:** Moreover, LLM-based sentiment scoring is not necessarily robust, introducing noise that could mislead downstream models, and for this work, we did not validate the LLM scores against human-coded ground truth.
