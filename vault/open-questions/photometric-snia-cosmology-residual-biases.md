title: Unresolved Biases in Photometric SNIa Cosmology
---

**Background:** Fully photometric cosmological analyses combining Type Ia supernova light curves and host galaxy photometric redshifts exhibit persistent parameter biases that require further algorithmic refinement and validation.

**Question / Future Work:** Investigate and mitigate the residual biases observed in recovered cosmological parameters (such as w and w0-wa) when performing fully photometric Type Ia supernova analyses combining deep-drilling field host photometric redshifts, photometric classifiers, and BEAMS with Bias Corrections (BBC). Future work needs to focus on incorporating more sophisticated host galaxy photometric redshift algorithms, properly integrating photometric redshift error distributions or full PDFs directly into the BEAMS/BBC framework, and modeling missing systematic uncertainties such as astrophysical models of intrinsic scatter, training set limitations of SED models like SALT3, and classifier-induced systematics.

**Why It Matters:** Addressing these residual biases is critical for ensuring that upcoming Rubin Observatory LSST cosmology analyses achieve systematic uncertainties competitive with statistical precision without biasing dark energy constraints.

**Evidence:** Averaging analysis results over 25 independent samples, we find small but significant biases indicating a need for further analysis testing and development. ... The analysis combines photometric Type Ia classification with host-galaxy photometric redshifts. However, residual biases in recovered cosmological parameters indicate that further analysis development is needed. Potential improvements include better galaxy photo-z algorithm, and using the photometric redshift error (or PDF) in BBC.
