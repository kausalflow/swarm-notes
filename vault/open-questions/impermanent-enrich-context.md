**Background:** The current Impermanent benchmark is instantiated using univariate time series derived from GitHub event counts.

**Question / Future Work:** Enriching the forecasting tasks with auxiliary contextual information beyond the raw event counts would provide a more complex and realistic assessment of model generalization capabilities in non-stationary environments.
