title: Proving local stationarity condition
---

**Background:** Infinite memory models represent a challenging class of time-varying parameter models, where estimation methods frequently rely on high-level assumptions about the local stationarity of the data-generating process. Proving that specific infinite memory processes satisfy these high-level local stationarity conditions is a necessary step for the rigorous validity of asymptotic theory.

**Question / Future Work:** A necessary task for future research is to provide formal proofs that complex, realistic infinite memory processes satisfy standard definitions of local stationarity. Establishing these properties is fundamental to ensuring the validity of consistency and normality proofs for local polynomial estimators in time-varying parameter frameworks.

**Why It Matters:** This is fundamental to ensuring the validity of asymptotic theory for the proposed estimators when applied to a wide range of realistic, complex infinite memory time series.
