title: Multi-Cell Real-Traffic SBFD Scaling
---

**Background:** Sub-Band Full Duplex (SBFD) architectures rely on adaptive uplink (UL) and downlink (DL) resource splits to maximize spectrum efficiency in highly dynamic traffic conditions. Current predictive resource allocation frameworks are primarily evaluated using single-cell, simulated traffic models.

**Question / Future Work:** There is a need to evaluate predictive reinforcement learning-based SBFD resource allocation frameworks in multi-cell, multi-base station environments using empirical network traffic datasets rather than synthetic ones to validate robustness and scalability.

**Why It Matters:** Real-world deployment requires generalization across heterogeneous cells and interference patterns, which single-cell synthetic simulations fail to fully capture. Evaluating multi-agent systems is critical to managing inter-cell interference in dense network deployments.
