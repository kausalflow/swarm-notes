title: Dynamic Light Spanner Update Time

**Background:** The construction of dynamic spanners that maintain a light-weight graph structure under insertions and deletions in metric spaces with constant doubling dimension is a significant challenge in computational geometry.

**Question / Future Work:** Is it possible to design a dynamic algorithm that maintains a light-weight spanner for a point set under dynamic updates in doubling metrics with polylogarithmic update time?
