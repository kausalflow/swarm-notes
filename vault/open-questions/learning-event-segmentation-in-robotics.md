title: Learning Event Segmentation
---

**Background:** Effective long-horizon robotic manipulation systems require mechanisms to delineate distinct, temporally separated events that define task boundaries.

**Question / Future Work:** Learning the appropriate event segmentation from raw sensory data without explicit supervision remains an open problem for event-driven memory architectures.
