**Background:** The analysis of predictor performance in closed-loop control was conducted using the infinite-horizon LQR cost framework. However, performance in other control or learning paradigms remains unexplored.

**Question / Future Work:** Analyze predictor performance in closed-loop control using frameworks other than the infinite-horizon LQR setting studied, such as model-based reinforcement learning or other robust control objectives.
