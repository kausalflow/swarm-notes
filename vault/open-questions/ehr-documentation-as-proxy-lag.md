title: EHR Documentation Lag vs. True Onset
---

**Background:** The presented work uses documented clinical codes as a proxy for the true underlying disease process in patients.

**Question / Future Work:** The inherent limitation of using electronic health record (EHR) documentation as a label is that documented onsets often lag behind the actual biological onset of a disease. It remains unclear how to build models that can infer earlier, subclinical events or how to assess the optimality of current modeling practices relative to an idealized, real-time understanding of patient health trajectories.
