title: Model life table use
---

**Background:** The trajectory functions define a continuous model life table system in Tucker coordinates, which is a byproduct of the forecasting framework that can be used independently.

**Question / Future Work:** The trajectory functions can be used independently of the forecasting component to generate model life tables for populations lacking age-specific data or to serve as a reference standard for evaluating the plausibility of observed mortality schedules.
