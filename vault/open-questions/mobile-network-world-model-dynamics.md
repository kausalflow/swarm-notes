title: Modeling mobile network world dynamics
---

**Background:** World models have demonstrated potential for internalizing causal structures in dynamical systems, yet their implementation in the domain of mobile networks remains in nascent stages.

**Question / Future Work:** A critical open problem is the development of a mobile network world model that can effectively capture the high-dimensional, non-local dynamics between network configuration parameters and multi-cell traffic states in a way that is robust for large-scale deployments.

**Why It Matters:** Addressing this is essential for transitioning mobile network management from static forecasting to an actionable, digital-twin driven paradigm where operators can simulate and optimize network configurations without live-network risk.

**Evidence:** Existing world models lack customized modeling of network topology and spatio-temporal traffic dynamics, making the construction of a mobile network world model capable of learning network state-parameter dynamics an urgent open problem.
