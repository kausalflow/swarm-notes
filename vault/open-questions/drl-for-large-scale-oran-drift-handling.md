title: Scaling Drift Handling via DRL
---

**Background:** Traditional tabular reinforcement learning frameworks for drift detection and model maintenance often rely on discretized state representations, which struggle to scale across complex, continuous-time network traffic variations.

**Question / Future Work:** There is an unresolved need for scalable Deep Reinforcement Learning (DRL) approaches to handle model maintenance in large-scale network deployments where traffic patterns exhibit continuous and non-stationary dynamics, rendering simple tabular methods insufficient.

**Why It Matters:** Addressing the scalability of drift-handling agents is a fundamental requirement for deploying reliable ML models in real-world telecommunications infrastructure.
