title: Predicting SEP Intensity and Duration
---

**Background:** Current machine learning models for solar energetic particle (SEP) forecasting primarily focus on binary event occurrence, which is insufficient for comprehensive risk assessment of radiation hazards. Quantifying event magnitude and temporal evolution is necessary for actionable decision-making.

**Question / Future Work:** Future work must move beyond binary classification to multi-output frameworks that predict continuous metrics like SEP peak flux and event duration. This requires integrating continuous regression targets into multi-task learning architectures while managing the differential sensitivity of these parameters to solar event input data.

**Why It Matters:** Intensity and duration estimates are critical for assessing the actual severity of radiation impacts on space infrastructure and human safety.
