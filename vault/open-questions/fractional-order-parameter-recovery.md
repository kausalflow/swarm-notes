title: Recovering Fractional System Orders
---

**Background:** Fractional-order dynamical systems exhibit non-exponential error-growth patterns, often analyzed through data-driven error-growth diagnostics. However, these fitted parameters typically capture aggregate system behavior rather than intrinsic physical fractional orders.

**Question / Future Work:** Determining how to map effective Mittag-Leffler shape parameters—derived from forecast-error growth curves—back to the true physical fractional order of the underlying dynamical system remains a challenging inverse problem. This requires robust methodology to disentangle attractor geometry from intrinsic fractional dynamics.

**Why It Matters:** Establishing this link is essential for moving beyond pattern-matching toward accurate system identification and physical interpretation in fractional time series.

**Evidence:** The fitted Mittag–Leffler order should be interpreted as an effective shape parameter of the error-growth curve rather than as a direct estimate of the true system order, the recovery of which remains a more difficult inverse problem.
