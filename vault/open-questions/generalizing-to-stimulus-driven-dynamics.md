title: Generalizing to stimulus-driven dynamics
---

**Background:** BrainDyn currently focuses on forecasting resting-state neural dynamics, which are characterized as relatively self-sustaining signals with minimal external sensory input. Expanding generative models to encompass stimulus-driven dynamics, such as those generated by auditory or visual inputs, remains a significant challenge.

**Question / Future Work:** A key limitation of the current model is its inability to generate stimulus-driven neural dynamics, as it is primarily designed and trained for resting-state activity. Future work is required to adapt the framework to scenarios involving external inputs like movie-watching or audio, which would necessitate training the model in contexts where multiple neural units are simultaneously perturbed or driven by exogenous signals.

**Why It Matters:** The transition from modeling intrinsic resting-state activity to modeling stimulus-evoked or task-related dynamics is essential for creating a truly versatile virtual brain model. This is a critical next step for researchers seeking to use these generative frameworks for cognitive or task-based neuroscience applications.

**Evidence:** BrainDyn currently models resting dynamics, which are in some sense self-sustaining dynamics with little external sensory input. While it is possible to potentially generate stimulus-driven dynamics with strong visual or auditory components such as movie-watching or audio inputs using this type of model, it would have to be trained in a setting where several neuronal units are simultaneously perturbed, which the current model has not undertaken.
