title: Agent linguistic robustness evaluation
---

**Background:** Current benchmarks for agent evaluation in quantitative finance often lack diversity in natural language phrasing, masking potential brittleness in model performance.

**Question / Future Work:** Developing benchmarks that explicitly test linguistic robustness—via paraphrasing or intentional ambiguity—is essential to ensuring agent reliability in real-world user interaction.

**Why It Matters:** Linguistic robustness is a primary requirement for deploying reliable, user-facing AI agents in high-stakes domains like finance.
