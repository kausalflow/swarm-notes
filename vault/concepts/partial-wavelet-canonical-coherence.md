# Partial Wavelet Canonical Coherence

> *Auto-generated stub. Edit this file to add more details.*

A frequency-domain, scale-specific, and time-varying measure of direct association between multivariate signals adjusted for high-dimensional confounders.


## Why It Matters

It provides the first frequency-domain formulation of partial canonical correlation analysis for multivariate nonstationary time series.
