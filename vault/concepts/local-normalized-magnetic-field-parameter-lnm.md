# Local-Normalized Magnetic Field Parameter (LNM)

> *Auto-generated stub. Edit this file to add more details.*

A time-series parameter that quantifies short-timescale magnetic field variability relative to a local median to distinguish coherent magnetic structures.


## Why It Matters

Provides a quantitative, reproducible metric to identify magnetic cloud boundaries by isolating field strength variability from local background.
