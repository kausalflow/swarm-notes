# Density-Valued Vector Autoregression

> *Auto-generated stub. Edit this file to add more details.*

A framework for modeling the evolution of multivariate density functions by mapping them to Euclidean space and applying factor-augmented VAR.


## Why It Matters

It extends classical vector autoregression (VAR) to the functional setting of time-varying probability density functions, which is essential for analyzing distributional dynamics rather than point estimates.
