# Persistent Homology for Network Resilience Analysis

> *Auto-generated stub. Edit this file to add more details.*

A methodology for quantifying network structural resilience by applying persistent homology to time-varying snapshots of geospatial trade networks under historical perturbation.


## Why It Matters

It bridges topological data analysis with network infrastructure modeling to identify latent resilience phases that traditional aggregate indicators often miss.

## Evidence

> We apply persistent homology to the ORBIS Geospatial Network Model of the Roman World in order to quantify the structural resilience of the Eastern Mediterranean trade network.
