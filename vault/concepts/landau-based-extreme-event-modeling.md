# Landau-based extreme event modeling

> *Auto-generated stub. Edit this file to add more details.*

The use of the Landau distribution as a heavy-tailed alternative to GEV/GPD distributions for characterizing extreme value statistics in time series.


## Why It Matters

Provides a statistically superior alternative to traditional extreme value distributions for modeling heavy-tailed extreme event intensities in complex systems.

## Evidence

> the Landau distribution, originally introduced in plasma physics, accurately captures extreme precipitation at approximately 93% of global locations
