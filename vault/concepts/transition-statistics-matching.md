# Transition-statistics matching

> *Auto-generated stub. Edit this file to add more details.*

An operator-theoretic learning approach for dynamical systems that matches the motion of probability mass rather than individual point trajectories.


## Why It Matters

Provides an operator-theoretic alternative to trajectory-level losses for learning chaotic systems, improving robustness to noise and sparse data.
