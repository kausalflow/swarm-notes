# Generative Adversarial Learning from Deterministic Processes

> *Auto-generated stub. Edit this file to add more details.*

A theoretical framework for training generative adversarial models on single-trajectory time series generated by chaotic dynamical systems instead of i.i.d. samples.


## Why It Matters

Extends the theoretical foundation of GANs beyond the standard i.i.d. assumption to address learning from chaotic dynamical systems.

## Evidence

> We prove that it is possible, using an infinite-dimensional model of generative adversarial learning (GAL), to learn the invariant distribution of a sufficiently chaotic dynamical system from a single deterministically evolving time series
