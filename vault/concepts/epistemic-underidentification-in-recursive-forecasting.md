# Epistemic Underidentification in Recursive Forecasting

> *Auto-generated stub. Edit this file to add more details.*

The phenomenon where recursive autoregressive forecasting fails because self-generated states lack ground-truth labels and are not uniquely determined by observable history.


## Why It Matters

Reinterprets the long-standing 'exposure bias' problem in autoregressive forecasting as a structural epistemic failure rather than simple distribution shift.

## Evidence

> recursive rollout is also an epistemic underidentification problem
