# Hybrid System Embedding Theorem

> *Auto-generated stub. Edit this file to add more details.*

A mathematical result establishing that n-dimensional hybrid systems can be embedded into m-dimensional Euclidean space (m > 2n) as continuous vector fields.


## Why It Matters

Provides a theoretical foundation for representing discontinuous hybrid dynamics as continuous vector fields, enabling the use of standard differentiable optimization tools.
