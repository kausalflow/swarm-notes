# Anti-recurrence plot (anti-RP)

> *Auto-generated stub. Edit this file to add more details.*

A diagnostic technique for identifying antiperiodic symmetry in dynamical systems by measuring cross-recurrence between a trajectory and its point-reflected image.


## Why It Matters

Provides a novel model-free diagnostic for detecting dynamical symmetry (specifically sign-reversal invariance) that standard recurrence plots fail to identify.

## Evidence

> a complementary \\emph{anti-recurrence plot} (anti-RP), built from the cross recurrence between a trajectory and its point-reflected image, detects it directly.
