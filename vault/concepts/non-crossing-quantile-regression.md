# Non-Crossing Quantile (NCQ) Regression

> *Auto-generated stub. Edit this file to add more details.*

A regression head for time prediction that enforces monotonic ordering on quantile estimates to ensure stability under heavy-tailed temporal distributions.


## Why It Matters

It addresses the instability of mean-based time prediction in heavy-tailed inter-arrival distributions.
