# Generalised Network Heterogeneous Autoregressive Model (GNHAR)

> *Auto-generated stub. Edit this file to add more details.*

A multivariate volatility forecasting framework that extends the heterogeneous autoregressive (HAR) approach by incorporating cross-sectional spillovers via a directed graph.

It is the core methodological contribution, providing a parsimonious multivariate framework for volatility forecasting that accounts for cross-sectional market dependencies.
