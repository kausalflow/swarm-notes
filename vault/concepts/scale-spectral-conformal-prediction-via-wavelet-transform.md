# Spectral Conformal prediction via wAveLEt transform (SCALE)

> *Auto-generated stub. Edit this file to add more details.*

A conformal prediction framework that decomposes graph time series into frequency components using wavelets to perform adaptive conformalization on high-frequency residuals.


## Why It Matters

It is the primary method proposed to realize valid coverage and improved coverage-efficiency for multivariate graph time series.

## Evidence

> SCALE uses graph wavelets to decompose low/high-frequency components and conformalizes high-frequency residuals via adaptive gating over a low-frequency embedding.
