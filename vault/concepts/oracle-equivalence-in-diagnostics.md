# Oracle-Equivalence in Diagnostics

> *Auto-generated stub. Edit this file to add more details.*

A property of diagnostic procedures where residuals computed using estimated parameters exhibit the same limiting behavior as if the true parameters were known.


## Why It Matters

This property allows for diagnostic statistics that avoid the complexities of parameter-estimation effects, enabling more efficient bootstrapping.

## Evidence

> residual-based process has the same limiting null distribution as the infeasible oracle process based on the true errors
