# Online Variance-Based Active Learning (OVBAL)

> *Auto-generated stub. Edit this file to add more details.*

A cost-aware online active learning framework that uses D-optimality and exponential forgetting to adapt to nonstationary streams under rolling budget constraints.


## Why It Matters

Provides a unified framework for balancing label cost, model utility, and budget constraints in nonstationary data streams.

> we propose OVBAL (online variance-based active learning), which integrates data pricing with information-driven selection by estimating each sample's marginal utility via a D-optimality criterion with exponential forgetting
