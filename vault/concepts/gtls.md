# GTLS

> *Auto-generated stub. Edit this file to add more details.*

A GPU-accelerated implementation of the Transit Least Squares (TLS) algorithm for efficient periodic transit detection in astronomical light curves.


## Why It Matters

It addresses the critical computational bottleneck of searching for planetary transits in high-cadence astronomical photometric surveys by providing a GPU-accelerated alternative to the standard CPU-based Transit Least Squares (TLS) algorithm.

## Evidence

> We present Graphics Processing Units Transit Least Squares (GTLS), a GPU-accelerated implementation of the Transit Least Squares algorithm designed to reduce the computational cost of periodic transit detection while preserving TLS-like sensitivity to transit-shaped signals.
