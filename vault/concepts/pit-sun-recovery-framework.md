# Probability-Integral-Transformed Unbiased recovery (PIT-SUN)

> *Auto-generated stub. Edit this file to add more details.*

A framework that uses empirical marginal tables and multiplicative recovery to maintain expectation consistency when performing regression with nonlinear target transforms.


## Why It Matters

It solves the fundamental regression problem of achieving expectation-consistent estimation when using non-linear target transformations for heavy-tailed or zero-inflated data.
