# Inpainting-Style Time Series Forecasting

> *Auto-generated stub. Edit this file to add more details.*

A forecasting paradigm that treats future time-series values as missing regions to be reconstructed via masked generative processes like diffusion.


## Why It Matters

It provides a paradigm shift by reframing sequence generation as a constrained completion/inpainting task, enabling the application of image-based generative models like DDPM to time series.

## Evidence

> A key contribution of this work is the formulation of solar forecasting as an inpainting problem, where future time steps are treated as missing regions to be reconstructed.
