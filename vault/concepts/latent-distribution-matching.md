# Latent Distribution Matching

> *Auto-generated stub. Edit this file to add more details.*

A framework that unifies SSL methods by balancing latent alignment and entropy maximization.


## Why It Matters

Provides a unifying theoretical framework for diverse self-supervised learning methods, including contrastive, non-contrastive, and predictive approaches.

## Evidence

> We cast SSL as latent distribution matching (LDM): learning representations that maximize their log-probability under an assumed latent model (alignment), while maximizing latent entropy to prevent collapse (uniformity).
