# Non-Gaussian Diffusion Processes

> *Auto-generated stub. Edit this file to add more details.*

A framework for diffusion generative models using state-dependent diffusion coefficients and generalized Tweedie's formulae instead of standard Gaussian perturbations.


## Why It Matters

This is the core theoretical contribution, providing a formal bridge between classical stochastic calculus and modern score-based generative modeling for non-Gaussian regimes.

## Evidence

> we extend Tweedie's formula to important non-Gaussian processes, including geometric Brownian motion (GBM), squared Bessel (BESQ) processes, and Cox-Ingersoll-Ross (CIR) processes
