# Salience-based Temporal Editing

> *Auto-generated stub. Edit this file to add more details.*

## Why It Matters

It shifts the adversarial paradigm for time-series handwriting from additive noise to trajectory-preserving editing, addressing the problem of unnatural artifacts in previous methods.
