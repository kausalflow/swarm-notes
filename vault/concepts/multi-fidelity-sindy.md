# Multi-Fidelity SINDy

> *Auto-generated stub. Edit this file to add more details.*

A generalization of the SINDy framework that incorporates fidelity-weighted measurements through a generalized least squares formulation to improve nonlinear dynamical system discovery from inhomogeneous data.


## Why It Matters

It introduces a formal weighting strategy for SINDy to handle heteroscedastic noise in multi-fidelity data sources, which is a major limitation of standard SINDy.
