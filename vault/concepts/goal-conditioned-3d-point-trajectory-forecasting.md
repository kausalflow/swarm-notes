# Goal-Conditioned 3D Point Trajectory Forecasting

> *Auto-generated stub. Edit this file to add more details.*

A motion forecasting paradigm where future 3D point trajectories are predicted conditioned on visual history and natural language goal descriptions.


## Why It Matters

Establishes a unified, class-agnostic, and view-stable representation for motion forecasting that leverages language instructions, moving beyond standard pixel-based or class-specific approaches.

## Evidence

> We formalize the task of goal-conditioned 3D point motion forecasting: given a short visual history, a set of 3D query points on an object of interest, and a language description of the intended goal, the model predicts the future 3D trajectory of each point.
