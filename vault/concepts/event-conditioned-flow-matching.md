# Event Conditioned Flow Matching

A continuous-time generative model that synthesizes task-specific fMRI signals from resting-state data by conditioning on task event schedules through learned vector fields.


## Why It Matters

The core methodological contribution is the adaptation of flow matching for task-evoked fMRI synthesis by conditioning on heterogeneous event schedules.
