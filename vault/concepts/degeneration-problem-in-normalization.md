# Degeneration Problem in Normalization

> *Auto-generated stub. Edit this file to add more details.*

A phenomenon where gradient-based joint training of non-linear normalization parameters causes the model to collapse back to linear Z-score normalization.


## Why It Matters

Identifies a fundamental obstacle in training non-linear normalization layers alongside backbones, where parameters collapse to linear scaling.
