# Function-Valued Causal Influence

> *Auto-generated stub. Edit this file to add more details.*

A causal discovery framework that represents influence as state-dependent functions rather than scalar scores.


## Why It Matters

It shifts causal discovery from scalar edge scores to regime-dependent functions, overcoming information bottlenecks inherent in current nonlinear time series models.

## Evidence

> We formalize function-valued causal influence for additive, contribution-decomposable architectures and show that scalar causal scores constitute a severe information bottleneck
