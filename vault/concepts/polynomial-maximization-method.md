# Polynomial Maximization Method (PMM)

> *Auto-generated stub. Edit this file to add more details.*

An estimation method using third and fourth standardized moments to achieve higher asymptotic efficiency than ordinary least squares in the presence of non-Gaussian error distributions.


## Why It Matters

PMM provides a robust alternative to OLS for non-Gaussian, asymmetric, or leptokurtic errors by leveraging higher-order cumulants.
