# Variational Latent Basis Modeling (VLBM)

> *Auto-generated stub. Edit this file to add more details.*

A latent forecasting framework that decomposes multivariate time series into a shared low-rank subspace for stable dynamics and orthogonal residuals for OOD-induced deviations.


## Why It Matters

Provides a principled latent-space approach to decoupling stable temporal dynamics from OOD-driven residuals, improving robustness against distribution shifts.
