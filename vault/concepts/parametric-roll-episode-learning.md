# Parametric Roll Episode Learning

> *Auto-generated stub. Edit this file to add more details.*

A learned functional mapping designed to explicitly model and reproduce rare, high-consequence regime shifts in vessel response time series, such as those caused by parametric roll instability.


## Why It Matters

Parametric roll is a high-consequence, non-linear instability event in naval dynamics that this model is specifically trained to capture, which is a significant task beyond standard time-series forecasting.
