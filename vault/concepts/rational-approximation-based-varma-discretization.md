# Rational Approximation-Based VARMA Discretization

> *Auto-generated stub. Edit this file to add more details.*

A discretization technique using rational approximations in time to transform fractional stochastic partial differential equations into computable VARMA processes.


## Why It Matters

It provides a scalable computational framework for non-separable spatio-temporal models by relaxing restrictions on temporal smoothness.

## Evidence

> We propose a discretization method based on rational approximations in time to handle arbitrary smoothnesses, which leads to a vector autoregressive moving average process (VARMA).
