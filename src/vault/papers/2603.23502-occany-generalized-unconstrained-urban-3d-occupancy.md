# OccAny: Generalized Unconstrained Urban 3D Occupancy

## Summary

OccAny addresses the generalization limitations of existing 3D urban occupancy prediction models by creating the first unconstrained framework capable of handling out-of-domain, uncalibrated sensor inputs (monocular, sequential, or surround-view). The method introduces two key technical components: Segmentation Forcing, which leverages mask predictions to improve volumetric quality, and a Novel View Rendering pipeline for geometry completion via test-time augmentation. Experiments show OccAny surpasses visual geometry baselines while maintaining competitiveness against specialized in-domain self-supervised models on standard urban occupancy benchmarks. This work successfully bridges general visual geometry capabilities with the specific requirements of metric and semantic prediction in complex urban environments.

## Key Contributions

- OccAny is the first generalized 3D occupancy framework capable of operating on out-of-domain, uncalibrated urban scenes to predict metric occupancy and segmentation features.
- Introduction of Segmentation Forcing to improve occupancy prediction quality and enable mask-level output.
- Development of a Novel View Rendering pipeline that infers geometry for geometry completion through test-time view augmentation.
- OccAny outperforms existing visual geometry baselines and remains competitive with in-domain self-supervised methods across various input settings on established urban datasets.

## Key Concepts

- [Segmentation Forcing](../concepts/segmentation-forcing.md): A novel technique incorporated into the OccAny framework that enhances occupancy quality by enabling mask-level prediction alongside volumetric output.
- [Novel View Rendering pipeline](../concepts/novel-view-rendering-pipeline.md): A component within OccAny designed to infer novel-view geometry, which is then utilized for geometry completion via test-time view augmentation.
