# Dynamic Light Spanners in Doubling Metrics

## Summary

This paper addresses the challenge of dynamically maintaining a $(1+\varepsilon)$-spanner for a point set $X$ undergoing insertions and deletions within a metric space of constant doubling dimension. The authors introduce an efficient algorithm where the total weight of the resulting spanner is kept within a constant factor of the Minimum Spanning Tree (MST) weight. Crucially, both insertion and deletion operations are processed in polylogarithmic time with respect to the aspect ratio $\Phi$ of the point set. This work resolves an open problem by providing the first efficient dynamic algorithm for maintaining light-weight spanners, even in settings like low-dimensional Euclidean space.

## Key Contributions

- Developed a dynamic algorithm for maintaining a $(1+\varepsilon)$-spanner for a point set undergoing insertions and deletions in a metric space with a constant doubling dimension.
- The total weight of the maintained spanner is bounded by a constant factor of the weight of the minimum spanning tree (MST) of the point set.
- Each update (insertion or deletion) is performed in $\operatorname{poly}(\log \Phi)$ time, where $\Phi$ is the aspect ratio of the point set.
- This result provides the first efficient dynamic algorithm for maintaining a light-weight spanner, previously unknown even for low-dimensional Euclidean spaces.

## Limitations

The update time complexity depends on the aspect ratio $\Phi$, which can be large. Further research might explore complexity independent of $\Phi$.
