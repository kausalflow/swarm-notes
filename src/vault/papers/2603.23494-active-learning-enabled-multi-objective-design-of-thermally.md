# Active learning-enabled multi-objective design of thermally conductive and mechanically compliant polymers

## Summary

This work introduces an active learning (AL) framework leveraging multi-objective Bayesian optimization (MOBO) to accelerate the discovery of polymers optimized for both high thermal conductivity (TC) and mechanical compliance (low bulk modulus). The methodology trains independent Deep Kernel Learning (DKL) models as surrogate predictors for these properties and their uncertainties, using an initial dataset generated by a high-throughput molecular dynamics (MD) pipeline. The framework iteratively uses the parallel noisy expected hypervolume improvement (qNEHVI) acquisition function to recommend the most informative candidates for further MD simulation and model updating. This approach successfully identified six Pareto-optimal polymer candidates and provided insights into the underlying molecular feature trade-offs, significantly streamlining the development cycle for multifunctional materials.

## Key Contributions

- Developed an Active Learning (AL) framework powered by Multi-Objective Bayesian Optimization (MOBO) to simultaneously optimize for high thermal conductivity (TC) and low bulk modulus in polymers.
- Trained independent Deep Kernel Learning (DKL) surrogate models on initial molecular dynamics (MD) data to predict properties and estimate uncertainty, effectively addressing data scarcity.
- Utilized the parallel noisy expected hypervolume improvement (qNEHVI) acquisition function to efficiently explore the design space, leading to the identification of six candidates on the Pareto front.
- Provided interpretability analysis linking specific molecular features to the observed TC-modulus trade-offs and confirmed the practical relevance via synthesizability assessment.

## Key Concepts

- [Active Learning for Multi-Objective Optimization](../concepts/AL-enabled-MOBO-polymer-design.md): An active learning framework based on multi-objective Bayesian optimization (MOBO) that iteratively selects candidate polymers for molecular dynamics (MD) simulation to optimize for both high thermal conductivity and low bulk modulus.
- [Deep Kernel Learning Surrogate Models](../concepts/DKL-surrogate-models.md): Independent surrogate models trained using Deep Kernel Learning (DKL) to predict the thermal conductivity and bulk modulus of polymers, incorporating uncertainty quantification.
- [Parallel Noisy Expected Hypervolume Improvement (qNEHVI)](../concepts/qNEHVI-acquisition-function.md): The acquisition function used within the MOBO framework to guide the systematic screening and recommendation of new polymer candidates for evaluation.

## Limitations

The study relies on MD simulations for ground truth evaluation, meaning the accuracy is intrinsically limited by the fidelity of the MD force fields used. Future work should validate selected candidates experimentally.
